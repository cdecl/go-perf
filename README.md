
## go-perf
Performance metrics

#### Get (downloads the packages)

```sh
make dep 
```

#### Build 

```sh
make build
```

#### Usage 
```sh
go-perf.exe
Usage of go-perf.exe:
  -h string
        Elasticsearch host (require) e.g. http://localhost:9200
  -i string
        Elasticsearch index name (require)
  -t int
        Interval : default :20 (seconds) (default 20)
```
